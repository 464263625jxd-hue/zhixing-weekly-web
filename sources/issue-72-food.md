# 食品饮料板块（第72期）

盒马推出「有一层奶皮子的鲜牛奶」，元气森林上新咖啡新品「冰咖」...丨一周热闻

## 新产品

1. **瞄准非均质工艺！盒马上新「有一层奶皮子的鲜牛奶」**
近日，盒马上新「有一层奶皮子的鲜牛奶」。新品选用优质生牛乳，通过非均质工艺保留牛奶的鲜活营养与浓醇口感，静置冷藏3至5天后，牛奶顶部会自然形成厚层奶皮。配料仅为生牛乳，每100ml含有4g原生乳蛋白。目前售价17.9元/瓶。

2. **聚焦女性养颜需求！九阳豆浆推出「好看豆浆」系列新品**
近日，九阳豆浆上新「好看豆浆」系列，包括"无花果葡萄籽 好看豆浆"和"巴西莓烟酰胺 好看豆浆"两款新品。其中"无花果葡萄籽 好看豆浆"蛋白质含量为27g/100g；"巴西莓烟酰胺 好看豆浆"蛋白质含量为25.5g/100g。目前售价20g*7条/39.9元。

3. **时隔2年再推咖啡新品！元气森林上新「冰咖」系列产品**
近日，元气森林推出「冰咖」系列新品，包括爆汁葡萄和爆汁冰橙两款新品。选用东南亚进口咖啡豆，添加经-196℃液氮锁鲜冷榨技术处理的柠檬浓缩汁，低糖0脂。截至2026年4月1日，该产品位列天猫"咖啡饮料新品榜"榜首。目前售价900mL*12瓶/79.9元。

4. **盒马限定上新！让茶推出「青梅沏龙井」无糖果茶新品**
3月23日，让茶推出盒马限定「青梅沏龙井」无糖果茶新品。新品选用浙江绍兴龙井茶（添加量≥4g/L），其中2026年明前龙井新茶添加量≥1g/L，100%真茶萃取，搭配选自安徽黄山的青梅浓缩汁。新品上架当天便在盒马门店及线上APP全线售罄。目前售价4.5元/瓶。

## 公司新闻

1. **营收同比增长31.80%！东鹏饮料发布2025年财报**
3月31日，东鹏饮料发布2025年年度报告。2025年，公司全年实现营业收入208.75亿元，同比增长31.80%；归母净利润44.15亿元，同比增幅32.72%。公司可统计总动销量突破百亿瓶。

核心大单品"东鹏特饮"全年实现营收155.99亿元；电解质饮料"补水啦"全年实现营收32.74亿元，同比增长118.99%。

2. **净利润同比增长16.6%！安琪酵母发布2025年财报**
3月30日，安琪酵母发布2025年年度报告，公司2025年实现营业收入167.29亿元，同比增长10.08%；归母净利润为15.44亿元，同比增长16.6%。

全年酵母及深加工收入119.49亿元（+10.09%），制糖13.39亿元（+5.80%）。国内外市场方面，安琪酵母全年国内收入98.05亿元（+4.08%），国外68.48亿元（+19.88%）。

3. **1500亿"新巨头"诞生！联合利华正式宣布食品业务与味好美合并**
3月31日，联合利华正式宣布食品业务与味好美合并，预计于2027年年中完成。交易完成后，双方将在香料、调味品、调味料、烹饪辅助品、酱料等品类形成互补的全球业务布局。

4. **净利润同比增长180.9%！鸣鸣很忙发布2025年财报**
3月31日，鸣鸣很忙发布2025年财报。公司门店商品交易总额（GMV）达935.69亿元，同比增长68.5%；实现营业收入661.70亿元，同比增长68.2%；净利润23.29亿元，同比增长180.9%。

报告期内，公司新增门店7554家，截至2025年末，旗下"零食很忙""赵一鸣零食"门店总数达21948家。

5. **GMV达315.8亿元！霸王茶姬发布2025年财报**
3月31日，霸王茶姬发布2025年财报。2025年，公司实现GMV 315.8亿元，同比增长7.2%；营收为129.1亿元，同比增长4%；经调整后净利润为19.1亿元。

第四季度，霸王茶姬门店净增115家，总门店数达到7453家。海外市场第四季度GMV达3.7亿元，同比增长84.6%，海外GMV连续三个季度同比增长超75%。

6. **Keurig Dr Pepper获得皮爷母企控股权，任命Rafael Oliveira领导咖啡分拆业务**
4月1日，美国软饮料巨头Keurig Dr Pepper和皮爷母企JDE Peet's联合宣布，KDP已完成对JDE Peet's 96.22%股份的要约收购。整合后的业务将分拆成两家独立的美国上市公司。
