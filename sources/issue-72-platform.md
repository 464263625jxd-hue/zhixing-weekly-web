# 平台板块（第72期）

京东团购开启试运营；抖音电商与即时零售合流；鸣鸣很忙2025年GMV超935亿元丨零售电商周报

## 平台动作

### 阿里巴巴

1. **4月1日起 淘宝闪购下调超期仓储成本**
淘宝闪购日前面向平台商家发布仓配费用新政策。本次减免政策将于2026年4月1日开始执行。根据新政，对符合一定销售与周转效率的货品，淘宝闪购将直接减免其超期仓储费。

具体来看，若货品满足以下任一条件，平台将免收超期仓储费：
- 近30天内，该货品在全国闪购仓的日均单仓销售子单量达到0.1单及以上；
- 近30天内，该货品日均单仓销售子单量大于0但低于0.1单，同时全国周转天数（不含采购在途）不超过60天。

对于未达到上述免收条件的商品，淘宝闪购也给出了更具弹性的特惠减收方案。若货品周转天数小于等于60天（含60天），超期仓储费在优惠期内将按0元/立方/天计费；若货品周转天数超过60天，则按3元/立方/天收费。

2. **淘宝闪购新专送模式将向全国推广**
4月1日起，淘宝闪购在前期试点基础上，扩大新专送模式覆盖范围，在20多城落地，接下来将逐步向全国推广。新专送模式主要围绕骑士群体在收入透明度、保障机制、体验和发展等方面的核心诉求进行优化。

3. **盒马旗下超盒算NB广州连开两店**
盒马旗下硬折扣社区超市品牌超盒算NB在广州同步开出两家门店，分别落子黄埔区及番禺区。超盒算NB单店面积约600至800平方米，商品数量约1500款，自有品牌商品销售占比达60%。

4. **飞猪：过去一财年来自淘宝的租车预订量同比增长近60%**
飞猪租车业务保持强劲增长态势，来自淘宝的租车预订量同比增长近60%，下单用户数同比增长40%。

### 京东

1. **京东团购在全国百城百万店开启试运营**
京东团购在全国首批百城百万餐饮门店开启试运营。目前，在京东APP"秒送"频道已经可以看到"美食团购"标识。

2. **京东家电家居新出"百亿双补"计划**
京东与海尔、美的等超100个品牌正式推出家电家居"百亿双补"计划，未来一年内共同投入100亿元专项补贴。

3. **京东超市与蓝月亮达成战略合作**
京东超市与蓝月亮签订战略合作协议，三年内实现蓝月亮在京东销售额达50亿元。

4. **京东超市酒类一季度销售同比增长超25%**
京东超市酒类整体销售额同比增速超过25%，其中京东自营同比增速达35%以上。

### 拼多多

1. **拼多多调整抽检规则：材质安全描述不符 将被顶格罚款处罚**
拼多多对《商品描述及质量抽检规则》进行了新一轮修订。最大变化是，当商家违规时，需要赔付消费者的金额计算标准从"固定比例赔付"转向了"区间裁量+限定条件从重执行"。

2. **拼多多力推"优惠联合营销"**
拼多多面向商家上线"优惠联合营销活动"，将原本分散在省钱月卡、大促消费券、百亿补贴、秒杀优惠券等多个场景中的平台优惠资源进行统一整合。

3. **拼多多推出"红屁屁无忧"服务**
拼多多首次发布"红屁屁无忧服务条款"，将于4月10日正式生效。针对母婴类目下的纸尿裤、拉拉裤等核心刚需产品，建立了一套涵盖"使用后过敏保障、24小时快速响应、最高百元违约赔付"的售后体系。
