# 跨境板块（第72期）

亚马逊Prime Day收费模式变更；TikTok Shop欧日月活用户破2亿；SHEIN限制商家自发货丨跨境电商周报

## 平台动向

### 亚马逊

1. **亚马逊Prime会员日大促收费模式变更：固定费用改"预付+抽成"**
亚马逊开放了2026年Prime会员日促销提报通道，并公布了全新的收费规则。今年会员日促销活动提报于3月24日开启，远早于往年的4月至5月。卖家参与活动的费用结构从固定收费变为"固定预付+销售额抽成"的模式。

2. **亚马逊4月17日起加收FBA燃油附加费**
4月3日亚马逊通知，受燃油与物流成本上涨影响，4月17日起北美站点FBA配送费加收3.5%附加费，欧洲站点加收1.5%。

3. **亚马逊巴林云计算设施再遭伊朗袭击**
亚马逊位于巴林的云计算设施在伊朗袭击中受损。

4. **亚马逊欧洲站KYC审核升级，查验法人信息**
亚马逊欧洲站启动严格KYC审核，新增法人户口本本人页强制提交要求，60日内未完成核验将关停店铺并冻结资金。

### TikTok

1. **TikTok Shop欧日月活用户突破2亿，跨境POP商家年增长7倍**
TikTok Shop在欧洲（法德意西）和日本的月度活跃用户已突破2亿，整体体量与美区基本相当。英欧日跨境POP商家年增长达到7倍，目前已有约30万商家、近200万达人以及1500家服务商入驻相关站点。

2. **纽约市解除政府设备TikTok使用禁令**
美国纽约市长宣布，自3月31日起恢复允许政府各部门在电子设备上使用TikTok。

### 其他平台

1. **SHEIN将于4月中上旬限制美区商家自发货功能**
SHEIN表示将于4月中上旬关闭部分美区商家自发货功能，仅支持"在线下单"模式履约。

2. **美国宠物电商平台Chewy 2025财年净销售额达到126亿美元**
2025财年全年净销售额达到126亿美元，同比增长6.2%。

3. **eBay多伦多发力智能搜索，加速AI电商布局**
eBay在多伦多新设办公室并招募智能搜索产品设计师、搜索AI高级产品经理等岗位。

4. **Ozon进军土库曼斯坦市场**
俄罗斯电商平台Ozon宣布在土库曼斯坦正式开展业务。

5. **OTTO财报：2025/26财年GMV达75亿欧元**
德国电商OTTO 2025/26财年GMV达75亿欧元同比增6%。

## 商家圈

1. **绿联科技：2025年NAS业务暴增213%、净利大涨52%**
绿联科技全年实现营业收入94.91亿元，同比增长53.83%；归母净利润达到7.05亿元，同比增长52.42%。

2. **九号公司：营收利润双双大涨**
九号公司2025年实现营业总收入达212.8亿元，同比增长49.9%；归母净利润17.6亿元，同比增幅达62.2%。

3. **雅迪2025年利润大涨128.8% 国际扩张重心放在东南亚**
雅迪控股收入约为370.08亿元，同比增长31.1%；公司拥有者应占利润同比增长128.8%至29.12亿元。

4. **子不语2025年营收46.6亿元，同比增40.2%**
子不语全年实现总收入46.602亿元，同比增长40.2%；股东应占净利润2.692亿元，增长78.6%。

## 跨境物流

1. **DHL完成欧洲枢纽升级**
DHL已完成欧洲枢纽运营升级，整合人工智能分拣系统，使包裹吞吐量同比增长12%。

2. **伊朗战争冲击，快时尚电商面临成本压力**
上海至鹿特丹海运价涨20%，东南亚至欧洲空运价涨25%，原油上涨推高涤纶等服装原料成本超25%。

## 政策与数据

1. **宁波海关落地跨境电商零售出口分单模式**
跨境电商零售出口"分单模式"在宁波国际邮件互换中心正式启动，成为浙江省内首个落地该模式的业务现场。

2. **去年韩国线上购物交易额创新高**
2025年韩国线上购物交易额达2720.398万亿韩元，同比增4.9%创历史新高。

3. **欧盟达成史上最大关税改革协议**
欧洲议会与欧盟理事会达成1968年以来最大规模关税法改革协议，新增非欧盟直邮包裹处理费。

4. **2026年美国宠物行业预计升至1650亿美元**
美国宠物用品协会发布2026行业报告显示，2025年美国宠物行业总支出达1580亿美元，同比增3.7%。

5. **Adobe：2026美国复活节消费预计达249亿美元**
Adobe预测，2026年复活节前美国节日相关线上消费将较年内均值大涨255%。
