## 📋 项目说明

这是知行周刊的可视化网页，使用 **Alpine.js + Tailwind CSS** 构建的单页应用（SPA）。

### 核心特点
- ✅ **单文件架构** - index.html + data.json
- ✅ **零构建工具** - 直接在浏览器运行
- ✅ **响应式设计** - 完美适配移动端
- ✅ **实时搜索** - 全局资讯搜索
- ✅ **多视图切换** - 门户/阅读器/行业/企业
- ✅ **资讯分享** - 一键复制链接
- ✅ **数据标准化** - 完整字段 + 元数据
